## الوصف باللغة العربية

**Attendix** هو نظام ذكي متكامل لإدارة الحضور ومراقبة الحشود داخل القاعات الدراسية والمدرجات، تم تطويره كمشروع تخرج لعام 2026 لتقديم حلول أتمتة ذكية بديلة للطرق التقليدية الكلاسيكية. يعتمد النظام على تقنيات الرؤية الحاسوبية (Computer Vision) والذكاء الاصطناعي لرصد الوجوه وتتبع الحركة لحظياً.

###  المميزات والخصائص التقنية (System Features)

* **رصد الحضور الفوري:** التعرف على الوجوه (Face Recognition) وتسجيل الحضور تلقائياً فور دخول القاعة.
* **تتبع وتحليل الحشود:** استخدام موديل **YOLOv8** وخوارزمية **DeepSORT** لعد الأفراد بدقة داخل المدرجات ومراقبة الكثافة.
* **منطق تتبع ذكي:** إضافة خوارزمية زمنية ذكية للتعامل مع حالات الخروج المؤقت وإعادة الدخول (Exit & Re-entry Logic) لضمان دقة البيانات ومنع التلاعب.
* **بنية برمجية مرنة:** تطبيق مبادئ البرمجة كائنية التوجه (OOP) بالكامل لبناء معمارية برمجية منظمة وقابلة للتوسع والتطوير المستقبلي.

## English Description

**Attendix** is an AI-powered smart attendance and crowd monitoring system developed as a 2026 Graduation Project. The framework leverages state-of-the-art Computer Vision algorithms to automate attendance tracking and monitor classroom density in real-time.

###  Tech Stack & Core Implementation

* **Object Detection & Tracking:** Powered by **YOLOv8** and **DeepSORT** for real-time people counting and layout analytics.
* **Facial Recognition:** Integrated dynamic Face Recognition algorithms for instant, automated user identification.
* **Intelligent Exit/Re-entry Logic:** Engineered time-bound state tracking mechanisms to intelligently handle temporary classroom exits and re-entries.
* **Architecture:** Fully built upon strict **Object-Oriented Programming (OOP)** principles to achieve a modular, highly scalable, and maintainable software ecosystem.
